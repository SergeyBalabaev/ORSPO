## Задача 031: Реализация математической функции 3️⃣

### Задание

Написать программу, которая вычисляет значение функции для целого числа `x`:

$
f(x) = \frac{x^2 + 5x + 6}{1 + x + x^2}
$

- Входные данные: целое число `x`.
- Выходные данные: вещественное число (double) — результат вычисления функции.

---

### Прототип функции

```c
double my_function(int x);
```
