В этом коде показано, что доступ к элементу массива в C можно записывать разными способами, но результат будет одинаковым.

Массив `x` содержит три числа.
Выражение `x[1]` - это обычный способ получить второй элемент массива.

Выражение `*(x + 1)` делает то же самое, но явно через указательную арифметику:
к адресу начала массива прибавляется смещение, и результат разыменовывается.

Выражение `1[x]` выглядит странно, но по смыслу полностью эквивалентно `x[1]`.
В языке C операция `a[b]` определяется как `*(a + b)`, поэтому порядок не важен.

Выражение `*(&x[1])` сначала берёт адрес элемента массива, а потом разыменовывает его обратно, возвращая то же значение.
