# Задание 94.6

Адресация и стек

## Файлы

- `main.s` — исходный код на GAS (AT&T-синтаксис)
- `Makefile` — для сборки программы

## Сборка и запуск

Сборка через Makefile:

```bash
make
```

Запуск:

```bash
./main.out
```

Проверка кода возврата:
```bash
echo $?
```

Измените программу следующем образом:
Поместите на стек значения 10 и 2.

Далее после их помещения на стек, выполните команду pop %rax

Используя адресацию, получите значения из стека и вычтите одно из другого. Результат поместите в код возврата
