# Задание 94.5

Работа стека

## Файлы

- `main.s` — исходный код на GAS (AT&T-синтаксис)
- `Makefile` — для сборки программы

## Сборка и запуск

Сборка через Makefile:

```bash
make
```

Запуск:

```bash
./main.out
```

Проверка кода возврата:
```bash
echo $?
```

## Задание 1

Определите, какой адрес находится в регистре rsp после помещения туда переменной. Для этого воспользуйтесь GDB

```bash
gdb --tui ./main

break 5
run
info registers rsp
x/gx $rsp
n
```
