## Задача 036: Функция для решения квадратного уравнения 3️⃣

### Задание

Написать программу, которая решает квадратное уравнение: $ ax^2 + bx + c = 0 $

- Функция должна принимать коэффициенты `a`, `b`, `c` и возвращать количество действительных корней.
- Если есть корни, функция записывает их через указатели `x1` и `x2`.

---

### Прототип функции

```c
int solve_quadratic(double a, double b, double c, double *x1, double *x2);
```
