# Task071 - Сравнение строк

## Задание
Реализуйте функции для сравнения строк. Необходимо реализовать функцию основного сравнения и вспомогательные функции для проверки равенства, а также определения, является ли первая строка меньше или больше второй.

Запустите тесты и проверьте корректность результатов.
