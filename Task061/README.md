# Task061 - Двухсвязный список

## Задание
Реализуйте структуру данных двухсвязный список (doubly linked list). Необходимо реализовать функции для создания узла, добавления элементов, прямого и обратного обхода списка, а также удаления всего списка.

Запустите тесты и проверьте корректность результатов.
