## Task038: Inline-функции 3️⃣

### Задание

Дан файл `inline_func.c`, в котором реализована inline-функция:

```c
__attribute__((always_inline))
inline int summ(int a, int b)
{
    return a + b;
}
```

1. Изменить main так, чтобы программа:
    - Считывала три целых числа с клавиатуры;
    - Вызывала функцию summ дважды для вычисления суммы трёх чисел;
    - Выводила результат на экран в формате: ``` Sum: <результат> ```
2. Функция `summ` не меняется, используется напрямую в `main`.

### Запуск тестирования

Для проверки работы функции выполнить команду:

```bash
make test
```

### Контрольные вопросы

1. Что такое inline-функция и чем она отличается от обычной функции?
2. Зачем использовать `__attribute__((always_inline))`?
3. Можно ли вызвать summ несколько раз с разными аргументами в одной программе?
4. Почему использование inline-функций может ускорить выполнение программы?
