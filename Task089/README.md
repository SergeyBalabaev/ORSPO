# Task089 - Ассемблер: Код возврата 4️⃣

Простейшая программа на ассемблере для Linux, которая завершает процесс с кодом возврата 42.

## Файлы

- `main.s` — исходный код на GAS (AT&T-синтаксис)
- `Makefile` — для сборки программы

## Сборка и запуск

Сборка через Makefile:

```bash
make
```

Запуск:

```bash
./main.out
```

Проверка кода возврата:
```bash
echo $?
```
Должно вывести 42


Изменим режим с 32 на 64 битный при сборке. Для этого перепишем код:

```
_start:
    movq $1, %rax
    movq $42, %rdi
    syscall                   
```


Объясните отличия между movl и movq, eax и rax.
