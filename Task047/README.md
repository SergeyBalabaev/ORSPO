## Task047: Длина наибольшей последовательности цифр 3️⃣

### Задание

Напишите программу на C, которая:

1. Считывает строку длиной не более 100 символов;
2. Находит **самую длинную последовательность подряд идущих цифр** в строке;
3. Выводит длину этой последовательности.
4. Если цифр нет — выводит `0`.

---

### Пример работы программы

```text
# Ввод:
abc1234def56
# Вывод:
4

# Ввод:
abc
# Вывод:
0

# Ввод:
a1b22c333d4444
# Вывод:
4
```

### Запуск тестирования

Для проверки работы функции выполнить команду:

```bash
make test
```
