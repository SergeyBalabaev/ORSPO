## Task046: Префиксные суммы 4️⃣

### Задание

Напишите программу на C, которая:

1. Считывает число `n` — количество элементов массива (1 ≤ n ≤ 100);
2. Считывает `n` целых чисел;
3. Вычисляет массив **префиксных сумм**, где каждый элемент `i` равен сумме всех элементов исходного массива с индексами от `0` до `i`;
4. Выводит массив префиксных сумм через пробел.

---

### Пример работы программы

```text
# Ввод:
5
1 2 3 4 5
# Вывод:
1 3 6 10 15

# Ввод:
4
5 -2 3 1
# Вывод:
5 3 6 7
```
