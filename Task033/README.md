## Задача 033: Среднее арифметическое неизвестного числа чисел 3️⃣

### Задание

Написать программу, которая вычисляет среднее арифметическое n чисел, введённых пользователем.

Входные данные:
- Сначала число `n` — количество чисел
- Затем `n` целых чисел

Выходные данные:
- Среднее арифметическое (вещественное число) с точностью 6 знаков после запятой

---

### Прототип функции

```c
double average(int n, int numbers[]);
```

### Запуск тестирования
Для проверки работы функции выполнить команду:

```bash
make test
```
