## Задача 027: Время суток 3️⃣

### Задание

По заданному количеству секунд, прошедших с начала суток, определить, какое **время суток** это соответствует:

- 00:00 – 05:59 — ночь
- 06:00 – 11:59 — утро
- 12:00 – 17:59 — день
- 18:00 – 23:59 — вечер

Входные данные: целое число n (0 ≤ n < 86400).
Выходные данные: строка с названием времени суток.

---

### Прототип функции

```c
const char* get_time_of_day(int n);
```

Для проверки работы функции выполнить команду:

```bash
make test
```
