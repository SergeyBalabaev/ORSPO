## Задача 035: Работа с argc / argv 3️⃣

### Задание

Написать программу, которая принимает через командную строку несколько целых чисел и выводит их сумму.  

- Используйте `argc` и `argv` для обработки аргументов командной строки  
- Если числа не переданы, вывести сообщение `"No numbers provided"`  

---

### Пример вызова программы

```bash
./main 1 2 3
```
### Результат:

```
Вывод: 6
```

### Запуск тестирования
Для проверки работы функции выполнить команду:

```bash
make test
```

1. Что такое argc и argv в функции main?
2. Почему в цикле для суммирования чисел мы начинаем с i = 1, а не с i = 0?
3. Что произойдет, если пользователь не передаст аргументы?
4.  Что произойдет, если аргумент командной строки не является числом?
