# Task001 — Компиляция программы из одного файла 🔍 

Скомпилируйте и запустите программу используя команду:

```bash
gcc main.c -o main
./main
```

где:

- `gcc` — название компилятора  
- `main.c` — название входного файла  
- `-o` — ключ, после которого указывается название выходного файла  

# Задание

* Используя команду `gcc -v`, определите версию компилятора gcc.

* Под какую архитектуру происходит сборка программы?

* Найдите во внешних источниках, какая максимальная допустимая версия языка C может быть скомпилирована с помощью этой версии компилятора.
