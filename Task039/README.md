## Task039: Работа с массивами 3️⃣

### Задание

Напишите программу на C, которая:

1. Считывает число `n` — размер массива (n ≤ 100);
2. Считывает `n` целых чисел и сохраняет их в массив;
3. Выводит сумму всех элементов массива.

---

### Прототип функции (если нужно)

```c
int array_sum(int arr[], int n);
```

### Пример работы программы

```text
Ввод:
5
1 3 5 7 9

Вывод:
Sum: 25
```
### Запуск тестирования

Для проверки работы функции выполнить команду:

```bash
make test
```
