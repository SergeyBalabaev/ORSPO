# Task080 — Выставление n-го бита в единицу

## Описание задания

Вам дан файл `bin_operations.c` с кодом. Данный файл используется в нескольких заданиях подряд, поэтому некоторые функции могут не понадобиться в текущем задании. Но так или иначе изучите его, попробуйте запустить код, поиграйтесь с присваиваемыми значениями в нем.

Основная задача: изучите функцию `setBit()`. Узнайте в чем ее смысл и что она делает. Какая логическая операция проходит здесь?
Объявите свою структуру типа `code` и заполните ее с помощью этой функции числом, которое расчитывается так: `N = 150 + <номер_рабочего_места>`
