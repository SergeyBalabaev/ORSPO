# Task082 — Инвертирование n-го бита

## Описание задания

Вам дан файл `bin_operations.c` с кодом. Данный файл используется в нескольких заданиях подряд, поэтому некоторые функции могут не понадобиться в текущем задании. Но так или иначе изучите его, попробуйте запустить код, поиграйтесь с присваиваемыми значениями в нем.

Основная задача: изучите функцию `switchBit()`. Узнайте в чем ее смысл и что она делает. Какая логическая операция проходит здесь?
Объявите свою структуру типа `code`, проинициализируйте ее числом 170. Далее с помощью функции `switchBit` сделайте в структуре число, которое расчитывается так: `N = <номер_рабочего_места> * 4`
