## Задача 030: FizzBuzz 3️⃣

### Задание

Напишите программу, которая выводит числа от 1 до n с условиями:

- Если число кратно 3 — вывести `Fizz`
- Если число кратно 5 — вывести `Buzz`
- Если число кратно 3 и 5 — вывести `FizzBuzz`
- Иначе — вывести само число

---

### Прототип функции

```c
void fizzbuzz(int n);
```
