## Task040: Массивы — найти любое повторяющееся число 3️⃣

### Задание

Напишите программу на C, которая:

1. Считывает число `n` — размер массива (n ≤ 100);  
2. Считывает `n` целых чисел и сохраняет их в массив;  
3. Находит любое число, которое встречается более одного раза;  
4. Если такое число есть, выводит его; иначе выводит `"No duplicates"`.  

---

### Пример работы программы

```text
# Ввод:
5
1 3 2 3 5
# Вывод:
Duplicate: 3

# Ввод:
4
1 2 3 4
# Вывод:
No duplicates
```

### Запуск тестирования

Для проверки работы функции выполнить команду:

```bash
make test
```
