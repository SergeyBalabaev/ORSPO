## Task044: Отсортировать массив из слов по возрастанию 3️⃣

### Задание

Напишите программу на C, которая:

1. Считывает число `n` — количество слов (n ≤ 50);  
2. Считывает `n` слов (каждое не длиннее 49 символов);  
3. Сортирует слова по возрастанию (лексикографически);  
4. Выводит отсортированные слова через пробел.  

---

### Пример работы программы

```text
# Ввод:
5
banana apple cherry date apricot
# Вывод:
apple apricot banana cherry date

# Ввод:
3
Dog Cat Mouse
# Вывод:
Cat Dog Mouse
```

### Запуск тестирования

Для проверки работы функции выполнить команду:

```bash
make test
```
