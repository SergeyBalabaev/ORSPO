# Task072 - Передача строки в функцию

## Задание
Реализуйте функции для работы со строками через функции обратного вызова (callback). Необходимо реализовать функции преобразования строк: конвертация в верхний и нижний регистр, разворот строки, а также общую функцию для применения произвольной функции к строке.

Запустите тесты и проверьте корректность результатов.
