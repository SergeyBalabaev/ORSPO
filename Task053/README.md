# Task053 - Копирование указателей

## Задание
- Что именно копируется при выполнении операции `p2 = p1` — значение или адрес?
- Почему разыменование `*p1` и `*p2` даёт один и тот же результат?
- Какой объект в памяти изменяется при выполнении операции `*p2 = 50`?
- Почему после изменения через `p2` меняется и переменная `value`?
- Сколько различных областей памяти участвует в этой программе: сколько переменных и сколько адресов?
- Попробуйте заменить строку `int *p2 = p1;` на `int *p2 = &p1;`. Скомпилируется ли программа? Если да — как изменится её поведение?
- Где в этой программе может появиться undefined behavior при неправильном использовании указателей?
- Есть ли в данной программе SEGFAULT? Если его нет — почему? В каких модификациях кода он может появиться?
- В чём принципиальная разница между копированием указателя и копированием значения?
