# Task067 - АВЛ-дерево / красно-черное

## Задание
Реализуйте структуру данных АВЛ-дерево (AVL tree). Необходимо реализовать функции для создания узла, вставки элементов с автоматической балансировкой, поиска значения, вычисления высоты и баланса узла, а также операции ротации (левая и правая).

Запустите тесты и проверьте корректность результатов.
