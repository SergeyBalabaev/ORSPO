## Задача 24. Сложение двух чисел 3️⃣

### Условие

Напишите программу на языке **C**, которая запрашивает у пользователя
два целых числа и выводит их сумму.

Программа должна:
- считать два числа со стандартного ввода;
- вычислить сумму чисел;
- вывести результат на экран.

### Прототип функции

Рекомендуется реализовать вычисление суммы в отдельной функции:

```c
int sum(int a, int b);
```

### Запуск тестирования
Для проверки работы функции выполнить команду:

```bash
make test
```
