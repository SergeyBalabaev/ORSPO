В программе создаётся массив `x` из пяти элементов.
Далее объявляется указатель `y`, который указывает на весь массив целиком, и указатель `z`, который указывает на первый элемент массива.

`x` при выводе в `printf` неявно преобразуется в указатель на первый элемент массива.
`x + 1` означает адрес второго элемента.

`*y` даёт доступ к массиву как к единому объекту, поэтому `*y` по смыслу эквивалентен `x`.
Выражение `*y + 1` сдвигается на один элемент внутри массива.

Указатель `z` указывает на первый элемент массива, поэтому `*z` — это значение первого элемента.
Выражение `*z + 1` увеличивает само значение, а не адрес.
