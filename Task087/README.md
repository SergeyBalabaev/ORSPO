# Task087 — Поиск строки в файле

## Задание

Используя весь багаж знаний и опыта, полученного за предыдущие задания, необходимо написать программу, которая ищет строку в созданном и указанном вами файле. 

Программа должна принимать на вход строку, которую нужно найти, а на выход выдает номер строки и колонки, где находиться начало искомой строки.

Если же ничего не было найдено, то должно возвращаться `-1 -1`
