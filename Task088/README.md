# Task088 — Контрольная сумма файла

## Основы

**Контрольная сумма (КС)** — некоторое значение, рассчитанное по набору данных путём применения определённого алгоритма и используемое для проверки целостности данных при их передаче или хранении. Также контрольные суммы могут использоваться для быстрого сравнения двух наборов данных на неэквивалентность: с большой вероятностью различные наборы данных будут иметь неравные контрольные суммы.

В случае с файлами контрольные суммы применяются для того, чтобы удостовериться в целостности файла, и по хорошему после каждого изменения файла КС должна быть новой и абсолютно уникальной.

Самым главным для алгоритма просчета КС является хеш-функция (задания с хеш-таблицей, помните?). Именно от нее зависит, насколько точно ваш алгоритм скажет об целостности файла и насколько он будет хорош и популярен.

## Задание

В этом задании вам необходимо написать программу, высчитывающую КС для выбранного файла. Итоговое значение КС файла должно выдаваться на выход программы в 16-ой системе исчисления.

Логику работы хеш-функции можете взять из реализации функции для хеш-таблицы (суммирование значений символов) или же найти другую реализацию в открытых источниках.

Чтобы доказать, что ваша программа корректно работает:
- подсчитайте КС от файла
- измените файл
- снова подсчитайте КС

Если КС изменился, то ваша программа работает корректно.
