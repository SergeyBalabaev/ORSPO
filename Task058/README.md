# Task058 - Объединить два списка

Даны два заголовка отсортированных связанных списков list1 и list2.

Необходимо реализовать функцию, которая объединяет эти два списка в один отсортированный связанный список.

- Новый список должен быть сформирован путем объединения существующих узлов из list1 и list2 (без создания новых узлов).
- Функция должна возвращать заголовок объединённого связанного списка.

Пример сигнатуры функции на C:

```c
struct ListNode* mergeTwoLists(struct ListNode* list1, struct ListNode* list2);
```

Требования:

- Списки уже отсортированы по возрастанию
- Результирующий список также должен быть отсортирован по возрастанию
- Использовать только существующие узлы; не создавать новые
