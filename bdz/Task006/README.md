# Task006 — Флаги компиляции (math) 🔍

Скомпилируйте программу, используя команды:

```bash
gcc main.c -o main
gcc main.c -lm -o main
./main
```

где:

* **`-lm`** — флаг для подключения математической библиотеки libm.so (link math)
---
## Задание 

* Скомпилируйте программу без флага `-lm`. Какая ошибка появилась?

* Добавьте флаг `-lm` и скомпилируйте программу снова. Успешна ли сборка?

* Запустите программу и проверьте её работу.

* Объясните, для чего нужен флаг `-lm` и почему без него возникает ошибка.
