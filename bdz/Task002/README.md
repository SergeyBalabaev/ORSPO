# Task002 — Компиляция программы из нескольких файлов 🔍 
Скомпилируйте и запустите программу, состоящую из нескольких файлов используя команды

```bash
gcc main.c lib.c -o main
./main
```

# Задание

* Скомпилируйте программу и убедитесь, что она работает.

* Попробуйте включить заголовочный файл `lib.h` несколько раз в файле `main.c` (добавьте директиву `#include "lib.h"` ещё раз).

* Скомпилируется ли программа теперь?

* Объясните, почему так происходит. Что можно добавить в заголовочный файл, чтобы избежать проблем при многократном включении?
