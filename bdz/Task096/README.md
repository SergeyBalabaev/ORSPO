# Task096 - Ассемблер: Вызов функции 4️⃣

## Файлы

- `main.s` — исходный код на GAS (AT&T-синтаксис)
- `Makefile` — для сборки программы

## Сборка и запуск

Сборка через Makefile:

```bash
make
```

Запуск:

```bash
./main.out
```

Проверка кода возврата:
```bash
echo $?
```

Изучите код программы. Обратите внимание на то, как передаются аргументы в функцию, как возвращаются из нее.
