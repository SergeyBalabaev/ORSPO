# Task011 — Коды ошибок и assert 🔍

## Переменная errno и функция strerror()

**`errno`** — переменная, объявленная в заголовочном файле `<errno.h>`, которая содержит целочисленный код последней ошибки. Она устанавливается только в случае возникновения ошибки при вызове функций стандартной библиотеки и системных вызовов.

**Важно:** `errno` не сбрасывается автоматически при успешном выполнении функции, поэтому анализировать её значение следует лишь тогда, когда функция явно сигнализирует об ошибке (например, возвращает `NULL` или `-1`).

**`strerror()`** — функция из `<string.h>`, которая принимает код ошибки и возвращает строку с человекочитаемым описанием данной ошибки.

**`perror()`** — функция из `<stdio.h>`, которая выводит сообщение об ошибке в `stderr`. Она принимает строку-префикс, добавляет двоеточие и описание последней ошибки (соответствующее текущему значению `errno`).
