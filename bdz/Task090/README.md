# Task090 - Ассемблер: Hello, MPSU! 4️⃣

Простейшая программа на ассемблере, которая выводит "Hello, MPSU!" в стандартный вывод и завершает процесс с кодом возврата 0.

## Файлы

- `main.s` — исходный код на GAS (AT&T-синтаксис)  
- `Makefile` — для сборки программы

## Сборка и запуск

Сборка через Makefile:

```bash
make
```

Запуск программы:
```
./main
```

Ожидаемый вывод:
```
Hello, world!
```

Исправьте программу таким образом, чтобы она выводила **полностью** ваши ФИО и номер группы.
