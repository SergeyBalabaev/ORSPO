# Task017 — Основы GDB 🔍

## Что такое GDB?

**`GDB (GNU Debugger)`** — это мощный отладчик для программ, написанных на `C`, `C++` и других языках. Он позволяет:

* Останавливать программу в любой точке

* Исследовать значения переменных

* Изменять данные во время выполнения

* Анализировать стек вызовов

* Находить причины ошибок (segmentation fault, бесконечные циклы и т.д.)

--- 

## Компиляция с отладочной информацией

```bash
gcc -g gdb_vals.c -o gdb_vals
```

где:

* `-g` — ключ, добавляющий отладочную информацию в исполняемый файл (имена переменных, строки исходного кода и т.д.)

## Запуск отладки

```bash
gdb gdb_vals
```

| Команда | 
| :--- |
| `list` | 
| `break 32` |
| `clear 32` |   
| `run` |     
| `info registers` | 
| `info locals` | 
| `ptype a` | 
| `print a` |  
| `print &a` |  
| `print sizeof(a)` |  
| `set var a = 512` |

