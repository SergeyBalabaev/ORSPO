# Task098 - Взлом программы 🔍 

Проанализируйте программу и "взломайте ее", добившись вывода на экран сообщения о выигрыше

1. Проверяем порядок следования байт
```bash
readelf secret -a
```

2. Анализируем ассемблерный код
```bash
objdump secret -d
```

3. Анализируем запущенную программу
```bash
gdb secret
```
