# Task023 — float vs double: точность вычислений 🔍

## Типы с плавающей точкой в C

|Тип	|Размер (байт)|	Значимые цифры|	Диапазон (приблизительно)|	Стандарт|
| :--- | :--- | :--- | :--- | :--- |
|`float`|	4|	~6-7 десятичных цифр|	±1.5×10⁻⁴⁵ … ±3.4×10³⁸|	IEEE 754 single-precision|
|`double`|	8	|~15-16 десятичных цифр	|±5.0×10⁻³²⁴ … ±1.8×10³⁰⁸|	IEEE 754 double-precision|
|`long double`|	10/16|	~18-19 десятичных цифр|	зависит от платформы|	IEEE 754 extended-precision|

## Представление в памяти
|Тип|	Знак (бит)|	Экспонента (бит)|	Мантисса (бит)|	Смещение экспоненты|
| :--- | :--- | :--- | :--- | :--- |
|`float`|	1|	8|	23|	127|
|`double`|	1|	11|	52|	1023|

## Компиляция и запуск

```bash
gcc -Wall -Wextra -Wpedantic -Werror precision.c -o precision -lm
./precision
```
