# Task005 — Поэтапная компиляция 🔍 
Скомпилируйте программу последовательно с различными флагами компиляции, используя команды:

```bash
gcc -E main.c -o prMain
gcc -S main.c -o coMain.s
gcc -c main.c -o asMain.o
gcc main.o -o main
./main
```

где:

- `-E` — остановка после этапа препроцессирования

- `-S` — остановка после этапа компиляции (создание ассемблерного кода)

- `-c` — остановка после этапа ассемблирования (создание объектного файла)

- `-o` — ключ для задания имени выходного файла

## Задание

* Выполните последовательно все команды, указанные выше.

* Что происходит на каждом из этапов компиляции?
