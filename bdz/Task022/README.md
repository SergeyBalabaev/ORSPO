# Task022 — Константность в C 🔍

## Константность в C

### Константные переменные

```c
const int a = 10;     // a нельзя изменить
int const b = 20;     // то же самое, const справа от типа
```

Для сборки используйте команду:  

```bash
gcc-13 -std=c2x definition.c
```

### Константные указатели

В `C` существует несколько комбинаций `const` с указателями:

| Объявление |  Что означает |
| :--- | :--- |
|`const int *p`|Указатель на константное целое (данные нельзя менять, указатель можно)|
|`int * const p`|Константный указатель на целое (данные можно менять, указатель нельзя)|
|`const int * const p`|Константный указатель на константное целое (ничего нельзя менять)|
|`int const *p`|То же, что `const int *p`|

## Правило чтения сложных объявлений

### Читайте справа налево

```c
int * const * const ** const ptrptrx = &ptrx;
```

### Разбор:

* `const` — самый правый — константный

* `**` — два уровня косвенности

* `const` — следующий — константный

* `*` — еще один уровень косвенности

* `const` — еще один константный

* `*` — последний уровень косвенности

* `int` — тип данных

**Это означает**: `ptrptrx` — это константный указатель на указатель на константный указатель на указатель на целое число.
