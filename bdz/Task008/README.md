# Task008 — Clang-format и Clang-tidy 🔍

## Установка инструментов

```bash
sudo apt-get install clang-format clang-tidy
```
где:

* **clang-format** — инструмент для автоматического форматирования кода

* **clang-tidy** — инструмент для статического анализа кода (поиск ошибок, проверка стиля)
---
## Основные команды

```bash
# Форматирование файла (флаг -i означает "in-place" — изменения сразу в файле)
clang-format -i main.c

# Запуск статического анализа со всеми проверками
clang-tidy main.c -checks=*

# Запуск статического анализа с указанием стандарта языка
clang-tidy main.c -- -std=c11
```
--- 

## Задание  

* Установите `clang-format` и `clang-tidy` с помощью команды выше.

* Выполните форматирование программы: `clang-format -i main.c`. Что изменилось в коде?

* Запустите статический анализ: `clang-tidy main.c -checks=*`. Какие ошибки и предупреждения найдены?

* Запустите анализ с указанием стандарта C11: `clang-tidy main.c -- -std=c11`. Появились ли новые предупреждения?

* Исправьте все найденные ошибки в программе.
