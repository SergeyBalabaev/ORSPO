# Task054 - Двухсвязный список 3️⃣

## Задание
Реализуйте двусвязный список (doubly linked list).

Необходимо реализовать функции для:

- создания нового узла списка
- добавления элемента в конец списка
- вставки элемента по заданному индексу
- удаления элемента по значению
- поиска элемента по значению
- прямого обхода списка с выводом элементов
- обратного обхода списка с выводом элементов
- освобождения всей памяти, занятой списком
