# Task003 — Работа ifdef/ifndef 🔍

Скомпилируйте и запустите программу, используя команду:

```bash
gcc main.c -o main
```

## Задание

* Соберите программу. Какой результат был напечатан?

* Удалите строку `#define X`. Какой теперь результат?

* Скомпилируйте программу с флагом `-E` и изучите, что произошло.

* Объясните, почему результат изменился.
