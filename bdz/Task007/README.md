# Task007 — Флаги -Wall -Wpedantic -Werror 🔍

Скомпилируйте программу по очереди со следующими флагами, используя команды:

```bash
gcc -Wall main.c -o main
gcc -Wpedantic main.c -o main
gcc -Wall -Wpedantic -Werror main.c -o main
```

где:

* **`-Wall`** — включает большинство предупреждений компилятора

* **`-Wpedantic`** — требует строгого соответствия стандарту C

* **`-Werror`** — превращает все предупреждения в ошибки

## Задание

* Скомпилируйте программу с флагом `-Wall`. Какие предупреждения появились?

* Скомпилируйте программу с флагом `-Wpedantic`. Какие предупреждения появились?

* Объясните назначение каждого из флагов.

* Соберите программу с обоими флагами и добавьте флаг `-Werror`. Что произошло? Для чего предназначен этот флаг?

* Исправьте программу так, чтобы она компилировалась без предупреждений и ошибок.
