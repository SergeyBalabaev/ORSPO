## Задача 034: Прогрессия 3️⃣

### Задание

Написать программу, которая вычисляет n-й член арифметической прогрессии.

Входные данные:
- `a1` — первый член прогрессии
- `d` — разность прогрессии
- `n` — номер члена, который нужно найти

Выходные данные:
- Значение n-го члена (double)

---

### Прототип функции

```c
double progression(int a1, int d, int n);
```
