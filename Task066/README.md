# Task066 - Бинарное дерево

## Задание
Реализуйте структуру данных бинарное дерево поиска (Binary Search Tree). Необходимо реализовать функции для создания узла, вставки элементов, поиска значения, поиска минимума и максимума, а также удаления всего дерева.

Запустите тесты и проверьте корректность результатов.
