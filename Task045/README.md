## Task045: Матрица из символов 3️⃣

### Задание

Напишите программу на C, которая:

1. Считывает два числа `rows` и `cols` — размеры матрицы (≤ 20);
2. Считывает один символ `ch`;
3. Генерирует матрицу `rows × cols` случайных символов из диапазона `a–z`;
4. Выводит матрицу построчно;
5. Вычисляет и выводит процент ячеек, содержащих символ `ch`, с точностью до целого числа.

---

### Пример работы программы

```text
# Ввод:
3 4
a
# Возможный вывод:
b a c d
a e a f
g h i j
Percentage of 'a': 25%
```
