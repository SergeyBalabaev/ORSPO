## Задача 028: Среднее число 3️⃣

### Задание

Даны три целых числа. Определить **среднее число** (не максимальное и не минимальное).

- Входные данные: три целых числа a, b, c
- Выходные данные: среднее число из трёх

---

### Прототип функции

```c
int middle_of_three(int a, int b, int c);
```

### Запуск тестирования
Для проверки работы функции выполнить команду:

```bash
make test
```
