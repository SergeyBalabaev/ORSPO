## Task041: Найти минимум и максимум в массиве 3️⃣

### Задание

Напишите программу на C, которая:

1. Считывает число `n` — размер массива (n ≤ 100);
2. Считывает `n` целых чисел и сохраняет их в массив;
3. Выводит минимальный и максимальный элемент массива;
4. Если массив пустой (n = 0) — выводит `0 0`;
5. Если массив состоит из одного элемента — выводит его дважды.

---

### Пример работы программы

```text
# Ввод:
5
3 7 2 9 5
# Вывод:
2 9

# Ввод:
1
42
# Вывод:
42 42

# Ввод:
0
# Вывод:
0 0
```
