## Задача 032: Палиндром 3️⃣

### Задание

Написать программу, которая проверяет, является ли введённая строка палиндромом (читается одинаково слева направо и справа налево).

Входные данные: строка длиной до 100 символов
Выходные данные: `"Yes"` если палиндром, `"No"` если нет

---

### Прототип функции

```c
int is_palindrome(const char *s);
```

### Запуск тестирования
Для проверки работы функции выполнить команду:

```bash
make test
```
