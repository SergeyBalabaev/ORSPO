## Задача 026: Секунды суток 3️⃣

### Задание

Идет n-ая секунда суток. Определить, сколько **полных часов** и **полных минут** прошло к этому моменту.

- Входные данные: целое число n (0 ≤ n < 86400).
- Выходные данные: количество полных часов и полных минут, прошедших с начала суток.

---

### Прототип функции

```c
void seconds_to_time(int n);
```
