## Task042: Массивы — удалить подряд одинаковые значения 3️⃣

### Задание

Напишите программу на C, которая:

1. Считывает число `n` — размер массива (n ≤ 100);
2. Считывает `n` целых чисел и сохраняет их в массив;
3. Удаляет подряд идущие одинаковые значения;
4. Выводит результат на экран через пробел.

---

### Пример работы программы

```text
# Ввод:
8
1 1 2 2 2 3 3 4
# Вывод:
1 2 3 4

# Ввод:
5
5 5 5 5 5
# Вывод:
5

# Ввод:
0
# Вывод:
```

### Запуск тестирования

Для проверки работы функции выполнить команду:

```bash
make test
```
