Адресная арифметика - это возможность выполнять арифметические операции над указателями.
Если указатель указывает на элемент массива, то прибавление числа к указателю сдвигает его не на байты, а на элементы соответствующего типа.

В программе создаётся массив `a` и указатель `p`, указывающий на его первый элемент.
Выражение `*p` даёт первый элемент массива.
Выражение `*(p + 1)` даёт второй элемент, а `*(p + 2)` — третий.

Затем указатель сдвигается на три позиции вперёд: `p = p + 3`.
Теперь он указывает на четвёртый элемент массива, и разыменование `*p` возвращает соответствующее значение.
