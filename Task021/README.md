# Task021 — Типы данных

## Описание задачи

В этой задаче вам предстоит работать с функцией сложения двух чисел. Программа содержит намеренную ошибку в реализации функции `sum`, которая должна быть обнаружена с помощью юнит-тестов.

## Тип int в C

| Характеристика |  Значение |
| :--- | :--- |
|Размер|Обычно `4` байта (`32` бита)|
|Диапазон|от `-2,147,483,648` до `2,147,483,64`7|
|Максимум|`INT_MAX` (определен в `<limits.h>`)|
|Минимум|`INT_MIN` (определен в `<limits.h>`)|

### Переполнение знаковых типов
В стандарте `C` переполнение знаковых целых типов (`signed integer overflow`) приводит к неопределенному поведению (`undefined behavior`). Это значит, что программа может:

* "Обернуться" (`wrap around`) как беззнаковые

* Дать непредсказуемый результат

* Упасть с ошибкой

* Сделать что-то еще

Поэтому тест `sum(INT_MAX, 1)` может вести себя по-разному на разных компиляторах и платформах.

* Запустите тесты снова:

    * Убедитесь, что все тесты проходят

    ```

    ```
