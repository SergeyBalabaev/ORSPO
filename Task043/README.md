## Task043: Массив из слов — найти все слова с определенной буквы 3️⃣

### Задание

Напишите программу на C, которая:

1. Считывает число `n` — количество слов (n ≤ 50);
2. Считывает `n` слов (каждое не длиннее 49 символов);
3. Считывает один символ `c`;
4. Выводит через пробел все слова, которые начинаются с буквы `c` (с учетом регистра);
5. Если таких слов нет — выводит `"No words"`.

---

### Пример работы программы

```text
# Ввод:
5
apple banana apricot cherry avocado
a
# Вывод:
apple apricot avocado

# Ввод:
3
Dog Cat Mouse
z
# Вывод:
No words
```

### Запуск тестирования

Для проверки работы функции выполнить команду:

```bash
make test
```
